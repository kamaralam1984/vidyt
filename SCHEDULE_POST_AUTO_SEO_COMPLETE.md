## 🔧 Technical Implementation

### SEO Generation Flow
```
1. User uploads video
   ↓
2. Filename extracted → Title auto-filled
   ↓
3. User edits title
   ↓
4. 800ms debounce waits for typing to stop
   ↓
5. API call to /api/youtube/title-seo
   ↓
6. AI returns: description, hashtags, tags, seoRankScore
   ↓
7. Form fields auto-populate with results
   ↓
8. Viral score displays with color coding
   ↓
9. User can toggle Rank 1 / Viral mode → Score recalculates
```

### State Variables Added
```typescript
seoRankScore: number | null       // 0-100 viral score
titleSeoLoading: boolean          // Shows while generating
rank1Mode: boolean                // true = Rank 1, false = Viral
titleSeoDebounceRef: Ref          // Debounce timer
titleSeoAbortRef: Ref             // Cancel previous request
titleSeoGenRef: Ref               // Track which generation is latest
```

### New Handlers
```typescript
useEffect(() => {
  // Auto-generate SEO when title changes
  // Debounced at 800ms
  // Calls /api/youtube/title-seo
  // Updates form fields
})
```
