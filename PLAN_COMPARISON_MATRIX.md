## 🔄 Plan Modification Workflow

### Edit Plan in Super Admin

```
1. Go to /admin/super
2. Click "Billing & Plans" → "Manage Plans"
3. Find plan to edit
4. Click "Edit" button
5. Modify:
   - Pricing
   - Features list
   - Feature flags
   - Usage limits
   - Display strings
6. Click "Save"
7. Changes apply to new users immediately
   Existing users keep current plan until renewal
```

### Create New Tier

```
1. Click "Create New Plan"
2. Fill all required fields
3. Test thoroughly before publishing
4. Announce to users
5. Monitor adoption
6. Adjust based on feedback
```

### Archive Plan

```
1. Go to Manage Plans
2. Find plan to remove
3. Click "Deactivate" (soft delete)
4. Plan still active for existing users
5. Not shown to new users
```
