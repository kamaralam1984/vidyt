# Schedule Post - Auto SEO Generation Feature

## Overview
When you upload a video to schedule a post, the system automatically generates SEO-optimized content including title suggestions, descriptions, hashtags, and viral score ranking - right in the Schedule Post form!

## How It Works

### 1. **Upload Video**
- Drag and drop or click to select a video file
- The filename is automatically used as the initial title
- Video is analyzed for SEO optimization potential

### 2. **Automatic SEO Generation**
As soon as you upload a video and confirm the title, the system:
- Analyzes the title for search optimization
- Generates a SEO-optimized description
- Creates trending hashtags (with # prefix)
- Calculates a **Viral Score** (0-100)

### 3. **Viral Score / SEO Rank**
The score shows your video's optimization level:
- **90+** 🔥 **VIRAL** - Excellent optimization for trending/ranking
- **70-89** ⭐ **GOOD** - Good optimization potential
- **Below 70** ❌ **LOW** - Needs improvement

## Features

### Auto-Fill Fields
When a video is uploaded, these fields auto-populate:
- ✅ **Title** - From filename (editable)
- ✅ **Description** - AI-generated SEO description
- ✅ **Hashtags** - Trending & relevant hashtags with # prefix
- ✅ **Tags** - Keywords for discoverability

### Generation Modes

#### 🎯 Rank 1 Mode (Default)
- Optimizes for Google/platform rankings
- Focus on SEO keywords
- Long-tail keyword targeting
- Best for: Discovery and organic reach

#### 📊 Viral Mode
- Optimizes for viral potential
- Focus on trending/engagement
- Emotional hook targeting
- Best for: Shares, views, engagement

**Toggle anytime** - Change between modes while editing, and the SEO will regenerate!

### Live Generation
- **Auto-generates on title change** - As you edit the title, new SEO is generated
- **Debounced** - Waits 800ms after you stop typing to avoid too many API calls
- **Real-time loading indicator** - See when SEO generation is in progress
- **Instant updates** - Description and hashtags update automatically

## Editable Auto-Generated Content

All auto-generated content is **fully editable**:
- Edit the auto-generated title - score recalculates
- Edit the generated description - your version is saved
- Modify hashtags - add your own or keep AI's suggestions
- Everything is customizable before posting

## Performance Optimization

### Smart Debouncing
- Waits 800ms after you stop typing
- Prevents API calls on every keystroke
- Reduces server load
- Shows loading indicator during generation

### Cancellation
- If you type again while generating, the old request is cancelled
- Latest request takes priority
- No wasted API calls

## API Integration

Uses the existing `/api/youtube/title-seo` endpoint:
```
POST /api/youtube/title-seo
{
  "title": "Your video title",
  "rank1Mode": true  // or false for viral mode
}

Response:
{
  "description": "AI-generated SEO description",
  "hashtags": ["keyword1", "keyword2", ...],
  "tags": "comma,separated,keywords",
  "seoRankScore": 85
}
```

## FAQ

### Q: Can I edit auto-generated content?
**A:** Yes! Everything is fully editable. Auto-generation is a starting point.

### Q: Do I need to upload a video for SEO generation?
**A:** Yes, for automatic generation. URL-only posts still work but without auto-SEO.

### Q: What if the viral score is low?
**A:** Try:
- Editing the title with different keywords
- Toggling between Rank 1 and Viral mode
- Being more specific in the title

### Q: Does the score matter?
**A:** It's a predictor! 90+ indicates good optimization, but your content quality matters most.

### Q: Can I use both modes?
**A:** Yes! Generate with Rank 1, see the score, then toggle to Viral mode for comparison.
